**Вывод по динамике ARPU / ARPPU (январь–июнь 2025)**

За период с января по июнь наблюдается снижение ключевых метрик монетизации:

ARPPU снизился с ~1400 до ~1000, что указывает на уменьшение среднего чека или смену состава платящих пользователей (например, рост доли покупок в низкомаржинальных категориях) или сезонность.

ARPU снизился с ~300 до ~150, что говорит об общем ослаблении монетизации активной аудитории.

Доля платящих пользователей сократилась с ~30% до ~20%, что свидетельствует о снижении конверсии активности в покупку.
